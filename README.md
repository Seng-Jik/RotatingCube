## 游戏说明

在本游戏中，共有6个不同的谜题。  
对于每个谜题，你需要旋转谜题方块直到方块和屏幕上的阴影重叠为止。  

你可以使用鼠标来旋转方块，或者使用键盘上的方向键来旋转方块。
